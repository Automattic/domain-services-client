# Class: [\Automattic](../namespaces/automattic.md)[\Domain_Services](../namespaces/automattic-domain-services.md)[\Response](../namespaces/automattic-domain-services-response.md)[\Domain](../namespaces/automattic-domain-services-response-domain.md)\Restore

## Summary:

Response of a `Domain\Restore` command

## Description:

- The restore operation runs asynchronously on the server.
- A success response indicates that the restore request is queued, not completed.
